# ⚡ EatWise — AI Nutrition Operating System

EatWise is a calorie and macro tracker that acts as a dedicated **"Nutrition OS"** for athletes. Instead of complex manual logging, it uses advanced AI to parse free-form text input and instantly calculate precise nutritional breakdowns.

## 🛠️ Tech Stack
* **Frontend:** React Native (Expo)
* **AI Engine:** Google Gemini 2.5 Flash API (Structured Outputs)
* **Backend & Sync:** Supabase Cloud Storage
* **Design Philosophy:** Cyber-terminal / Neo-brutalism UI

## 🦾 Core Features
* **AI Cognitive Parser:** Just type what you ate (e.g., *"3 boiled eggs and 100g chicken breast"*), and the AI automatically extracts calories, protein, fat, and carbs.
* **Daily Mission System:** Gamified tracking ensuring you hit your vital targets (like >100g protein).
* **Energy Core Dashboard:** Clear, bold data hierarchy focusing heavily on remaining calorie limits.
